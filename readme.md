# SimLab

This repo contains the code for my Master's project.

You can see all of the videos here: https://people.cs.uchicago.edu/~rorymb/videos/

Version 3 contains the final updates needed for my Master's presentation. Further updates may occur later, but are not currently planned.

---

The plan for this project was to show a proof-of-concept for factory-scale science laboratories in simulation, and demonstrate that experiments designed by AI (esp. large language models) can be validated entirely within simulation.
